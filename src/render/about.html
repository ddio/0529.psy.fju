關於本網站

輔大性侵事件發生至今，已經一年，或是對有些人而言，已經過了半個月。 這段期間，各種評論不斷，有人希望釐清是非對錯，有人希望理解原因與過程， 有人則希望讓處理爭議的各種路線，透過這次機會，提供給大眾檢視。

本網站的目標，是透過相關人員的陳述、對話（參見文末）來理解， 這些說法中，究竟回應了本次事件的重要資料（朱同學的第一篇公開文章）中的哪些爭議， 哪些起了頭，以及哪些尚未論及。 希望能藉此，稍微還原此次事件中參與者們，包含了當事人、工作小組，以及輔大心理系成員， 是如何理解這次事件。

本網站希望盡量以持平的態度引用討論資料，如果網友們覺得有哪些詮釋有誤的地方， 歡迎在文章底下留言，並說明理由、資料來源，我們會盡快與你討論，並更正錯誤。 另外，這網頁以當事人的對話釐清為主，各方筆戰的論點先不一一收錄，因為這工作太龐大啦， 若有興趣的網友，歡迎自行挖坑，本網站的原始碼與資料可參見 github 。
